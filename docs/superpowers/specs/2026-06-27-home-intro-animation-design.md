# 首页入场动画 — 设计文档

日期：2026-06-27
状态：已与用户确认效果（基于 v5 交互预览），待实现

## 1. 目标

为博客首页（`/`）增加一个入场动画：

- 屏幕中央出现一个药丸型长条，样式类似主页导航栏（圆角、`bg-background`、边框、阴影）。
- 药丸条上方紧贴一行标题文字 `Joshua's Blog`，与药丸作为一个整体一起出现、一起退场。
- 药丸条中间文字为 `Welcome to explore my blog.`。
- 药丸背后有围绕它的玻璃光晕，颜色与背景渐变同色（`#659EB9`），但更浓更亮。
- 短暂停留后，药丸条**沿直线垂直上移、左右对称展开**，变形为顶部导航栏；中间欢迎文字淡出、左侧品牌名与右侧菜单淡入；到顶瞬间有一次"水滴"式弹性回弹。
- 玻璃光晕同步**向上舒展并铺宽**，连续过渡为当前主页顶部向下的背景渐变（不凭空消失）。
- 与此同时，首页主体内容从下往上、由隐到显地丝滑渐显。
- 动画结束后进入正常页面。

## 2. 触发与频率

- 仅在首页 `/` 播放。
- 每个浏览器会话只播放一次：用 `sessionStorage`（key：`introPlayed`）标记，本次会话内再次回到首页不重播；新会话重新播放。
- `prefers-reduced-motion` 用户：跳过电影化动画，直接进入正常页面（或极短淡入）。
- 支持跳过：点击 / 滚动 / 按 Esc 可立即结束动画进入正常页面。

## 3. 架构与文件

`Header` 来自第三方包 `node_modules/astro-pure/components/basic/Header.astro`，**不直接修改**。

- 新增组件：`src/components/home/IntroAnimation.astro`
  - 包含：全屏遮罩 `overlay`、玻璃光晕 `glow`、标题 `introTitle`、药丸 `pill`（内含 `welcome` / `brand` / `navmenu` 三段文字）。
  - 自带 scoped 样式与 `@keyframes navJelly1`（单次水滴回弹）。
  - 自带一段 module `<script>` 编排时间线。
- 在 `src/pages/index.astro` 引入该组件（仅首页）。
- 防闪烁：在 `<head>` 或首屏渲染前用一段内联 `is:inline` 脚本，根据 `sessionStorage` + `prefers-reduced-motion` 立刻决定是否需要播放；若需要播放，立即给 `<body>` 加类名 `intro-playing`，在首屏绘制前就隐藏首页主体内容与真实导航栏，避免内容"闪一下"再被遮住。
- 颜色：光晕与背景渐变统一使用 `#659EB9`（与 `index.astro` 的 `highlightColor` 一致）。

## 4. 真实导航栏的交接方式（关键落地决策）

遮罩里的药丸是**纯视觉元素**，不是真实 `Header`。交接策略：

1. 动画期间真实 `Header` 与首页内容被 `intro-playing` 隐藏。
2. 药丸在遮罩内完成"上移 + 展开 + 变形 + 回弹"，视觉上已经是一条位于顶部、样式与导航栏一致的长条。
3. 动画收尾时移除 `intro-playing`：真实 `Header`（首页顶部为透明态，仅品牌文字 + 按钮）与首页内容显现；遮罩（含药丸）在极短时间（~150–200ms）内淡出移除。
4. 因为药丸末态的品牌文字位置、字号与真实 `Header` 顶部态的品牌文字对齐，遮罩淡出时下方真实导航栏的 `Joshua's Blog` 已在同一位置，交接连续、不突兀。这一步是"收尾移除遮罩"，与被否决的"药丸与导航栏交叉淡出变形"不是一回事——变形本身仍是真实的容器形变。

实现时需让药丸末态几何（顶部位置、内边距、品牌/菜单的左右位置）尽量贴合真实 `Header` 顶部态，保证收尾对齐。

## 5. 时间线（约 2.5s，基于 v5 预览）

| 阶段 | 时间 | 内容 |
|---|---|---|
| ① 淡入 | 0–0.6s | 标题 + 药丸 + 光晕一起淡入放大；药丸中间显示欢迎文字 |
| ② 停留 | 0.6–1.65s | 停留供阅读；光晕轻微呼吸 |
| ③ 变形 | 1.65–2.07s | 药丸沿直线上移（~0.42s 干脆缓动）、左右对称展开为导航栏；欢迎文字淡出、品牌/菜单淡入；标题淡出上移；光晕向上舒展铺宽、背景渐变同步淡入接管；首页内容从下往上渐显 |
| ④ 回弹 | 接 ③ 到顶瞬间 | 单次水滴回弹（`navJelly1`：一次纵向压缩 + 一次回弹归位），与到顶略微重叠，无空档 |
| ✓ 收尾 | ~2.5s | 遮罩淡出移除，进入正常页面 |

慢速模式仅用于预览调试，正式实现使用 1× 时长。

## 6. 三个关键效果的实现要点

- **药丸 → 导航栏（直线变形）**：药丸始终 `left:50%` + `transform:translateX(-50%)` 水平居中固定，仅动画 `top`（垂直上移）与 `width`（300px → `calc(100% - 36px)` 对称展开），并过渡 `height` / `border-radius` / `padding`。中心点走笔直竖线，左右对称展开。`transform` 在变形阶段保持不变，留给回弹关键帧使用。
- **水滴回弹（单次）**：到顶瞬间（用 `transitionend` 监听 `top`，或与变形时长对齐的定时器，略微提前以重叠）触发 `navJelly1` 关键帧：`scaleY` 先压缩到约 0.83、再回弹到约 1.06、最后归位 1。`transform-origin:center`。一次压缩 + 一次回弹，不来回抖动。
- **光晕舒展 → 背景渐变**：光晕为 `radial-gradient(#659EB9…)` + `filter:blur()`。变形阶段它向上移动、`width` 铺宽到约 150%、模糊度略增、透明度降为 0；同时全屏顶部向下的 `linear-gradient(#659EB9, transparent)`（即现有 `#highlight-gradient` 的等效背景）同步淡入接管。两者均为同色蓝，重叠过渡，蓝色全程连续，实现"舒展成背景渐变"而非消失。
- **首页从下往上渐显**：复用现有 `fade-in-up` 关键帧 / `.animate` 机制。用 `intro-playing` 在动画期间将首页主体 opacity 压为 0（并暂停或延迟其入场动画），动画结束移除类名时触发由下而上的淡入。

## 7. 健壮性

- **防闪烁**：首屏渲染前内联脚本决定是否播放并加 `intro-playing`。
- **无障碍**：`prefers-reduced-motion` 跳过电影化动画。
- **跳过**：点击 / 滚动 / Esc 立即结束并清理（移除遮罩、移除 `intro-playing`、写入 `sessionStorage`）。
- **深色模式**：药丸用主题色变量（`bg-background` / 边框 / 文字）；光晕 `#659EB9` 在深浅色下均协调。
- **不引入动画库**：纯原生 CSS keyframes + transition + 少量内联 JS，零额外依赖。
- **不修改 `node_modules`**：所有改动集中在 `src/components/home/IntroAnimation.astro`、`src/pages/index.astro`，必要时在 `src/assets/styles` 增补少量样式。

## 8. 范围之外（YAGNI）

- 不在非首页路由播放。
- 不做可配置的多套主题/文案后台。
- 不做声音、粒子等额外特效。
